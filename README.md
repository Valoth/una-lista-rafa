# Lista de Exercícios em Java - Estruturas de Controle

Este projeto reúne uma lista de exercícios desenvolvidos em Java com foco em lógica de programação, estruturas condicionais e estruturas de repetição.

A aplicação foi construída com um menu interativo utilizando `switch case`, permitindo ao usuário escolher qual exercício deseja executar.

## Objetivo

Praticar conceitos fundamentais da programação em Java por meio da resolução de problemas clássicos, envolvendo:

- Tomada de decisão (`if`, `else`, `switch`)
- Estruturas de repetição (`for`, `while`, `do while`)
- Entrada e saída de dados
- Validação de informações

## Aprendizados

Durante o desenvolvimento deste projeto, foram trabalhados conceitos essenciais:

- **Entrada de dados (`Scanner`)**  
Permite capturar informações digitadas pelo usuário no console.

- **Saída de dados (`System.out.println` e `printf`)**  
Utilizada para exibir informações e também formatar valores (como dinheiro).

- **Estruturas condicionais (`if/else` e `switch`)**  
Aplicadas para tomada de decisões e controle do fluxo do programa.

- **Estruturas de repetição (`for`, `while`, `do while`)**  
Usadas para repetir processos e validar entradas.

- **Tratamento de regras de negócio**  
Como divisão por zero, validação de categorias e limites de valores.

## Exercícios

### 1. Par ou Ímpar
Verifica se um número inteiro é par ou ímpar.

### 2. Maior de Dois
Compara dois números e informa o maior ou se são iguais.

### 3. Calculadora
Realiza operações básicas (soma, subtração, multiplicação e divisão), com tratamento de divisão por zero.

### 4. Validação de Entrada
Solicita uma nota entre 0 e 100, repetindo até que o valor seja válido.

### 5. Sentinela
Lê números até o usuário digitar 0 e informa o maior e o menor valor.

### 6. Simulador de Caixa
Sistema simples com depósito, saque e consulta de saldo, incluindo validações.

### 7. Desconto por Categoria
Aplica descontos com base na categoria do cliente (comum, premium ou funcionário).

### 8. Contagem de Valores
Lê 10 números e conta quantos são positivos, negativos e zeros.

### 9. Soma de 1 até N
Calcula a soma de todos os números de 1 até N.

### 10. Contagem Regressiva e Soma
Exibe a contagem regressiva de N até 1 e calcula a soma total.
